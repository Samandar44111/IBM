Overview

Build Your Perfect Busan Hike is an interactive tool that helps users discover the best hiking trails in Busan based on their preferences. The platform offers tailored trail recommendations considering factors like duration, difficulty, scenery, season, and transport access. With stunning design and smooth UX, it's a go-to resource for both beginners and serious hikers.

Problem Statement

There are many great hiking trails in Busan, but new hikers or tourists often struggle to choose the right trail for their needs. They may not know which mountains are scenic in each season, which are beginner-friendly, or how to reach them via public transport. Existing resources are often too general or lack interactivity.

Target Users

  • Travelers visiting Busan interested in nature and hiking
  • Locals seeking new trail experiences
  • Beginners looking for easier hikes with helpful guidance

Core Features

  • Interactive questionnaire that matches users to hiking trails
  • Top 3 recommendations with match percentage
  • Detailed mountain info: height, difficulty, duration, scenery, transport, best season, and unique highlights
  • Mobile-responsive UI with animations and smooth transitions

Stretch Goals / Extensions

  • Add a “favorites” or “bookmark” feature for saving trails
  • Allow users to submit photos or reviews of trails
  • Implement a multilingual version (e.g., Korean, Japanese, Chinese)
  • Include real-time weather updates for each trail
  • Offer GPX file downloads for trail navigation

Success Criteria

  • Fully functional site that matches users to hikes based on 5 preference factors
  • Includes data for all 10 featured mountains
  • UI is clean, interactive, and works on mobile
  • At least one stretch goal implemented (e.g., animations, responsive layout)
  • PRD and process documentation complete and clearly written

Technical Notes

  • Built entirely with HTML, CSS, and JavaScript – no external dependencies
  • Uses a simple scoring algorithm to rank hikes by user input
  • All content is self-contained and easily portable

Deliverables

  • Standalone HTML site (no external libraries or APIs)
  • Product Requirements Document (this page)
  • Live demo hosted on GitHub Pages
  • Process documentation in Google Docs
